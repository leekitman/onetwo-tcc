# onetwo-tcc
一个简单的tcc事务框架

## 要求
- JDK 1.8+
- Spring 4.0+
- RocketMQ 3.x + 
- Spring Boot 1.4.x, 1.5.x



## maven

**当前snapshot版本：0.5.1-SNAPSHOT**

若使用snapshot版本，请添加snapshotRepository仓储：

```xml
<repository>
     <id>oss</id>
     <url>https://oss.sonatype.org/content/repositories/snapshots/</url>
    <snapshots>
        <enabled>true</enabled>
    </snapshots>
</repository>   
```



其它依赖可以直接参考sample项目的pom.xml配置
